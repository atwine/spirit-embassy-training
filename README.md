# Spirit Embassy Training

A GitHub Pages site for Spirit Embassy / GoodNews Church training materials, teachings, and study resources.

## Current Course

- **The God-Kind Faith Accelerator Course** — Study Guide for Chapters 6 & 7
  - Live site: https://atwine.github.io/spirit-embassy-training/

### Site Structure

This site uses a two-page structure for registration gating:

- `index.html` — Landing page with Tally registration form link. The Tally form should redirect successful submissions to `study.html`.
- `study.html` — Full study content (Chapters 6 & 7) and the Tally quiz link.

Additional training materials and courses will be added to this repository over time.
